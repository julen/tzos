Ongi etorri
===========

Ongi etorri TZOSen dokumentazio teknikora. Orri hauetan zehar sistemaren
funtzionamenduari buruzko argibideak eta garapenaren inguruko xehetasun
teknikoak aurkituko dituzu. Instalatzeko gida ere badago.
